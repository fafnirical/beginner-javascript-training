Lesson 4 Exercises

For these exercises, you should follow the instructions in fafirical/typescript-starter. Push your code up to your fork when you're done, and submit the link to your GitHub repository.

  1. Modularize the code you've written in the previous exercises.

    • Start by breaking out your ATM class into a separate file, named "Atm.ts" (in the "src" folder). Export it from this file so you can use it elsewhere.
    • In your "index.ts" file, use (and test) the ATM class. Create some accounts, deposit some money, withdraw some money, etc. Go wild.
    • Create an interface for your ATM class in "AtmInterface.ts". Export it and have your ATM class (in "Atm.ts") implement it.
  2. Update your ATM class method to allow only "savings" and "checking" account types to be created. (Tip: you can use string literal types for this)
