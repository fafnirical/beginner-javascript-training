Lesson 2 Exercises

  1. Create an ATM class.

    It should have the following properties:

    • type (e.g., "checking"), which should be determined by some input
    • money, which should start at 0
    • transactionHistory, which should track of withdrawals and deposits made the account (indicate whether the transaction was a withdrawal or deposit)

    It should have the following methods:

    • deposit(), which should increase the money in the account by the input amount
    • withdraw(), which should decrease the money in the account by the input amount
    • checkBalance(), which should print the amount of money in the bank to the console
  2. Create the same ATM class from the previous exercise using object prototypes.

  3. (Bonus) Give the ATM class (or object prototype, if you prefer) some overdraft protection via a backupAccount property that can (optionally) contain a reference to another instance of the class, determined by input.

    • Whenever an ATM's balance would go below zero, it should trigger a withdrawal from the backup account equal to the overdraft.
