Lesson 3 Exercises

You can use the TypeScript Playground for this exercise.

  1. Using the ATM class that you created in lesson 2, convert it to TypeScript.

    In particular:

    • The type property should be a string
    • The money property should be a number
    • The transactionHistory property should be whatever you decided to use in lesson 2. If you're using an object, write a simple interface for it and use the interface

    The methods should also be typed.

    • The constructor should accept a string
    • deposit() should accept a number
    • withdraw() should accept a number
    • checkBalance() should have no parameters
